# :green_heart: AgendaBootcampEverisDio 

Atividade da aula 1 do curso **Utilizando recursos nativos do Android** integrante do bootcamp de Kotlin da everis em parceria com a Digital Innovation One.
Professor: Daniel Richter

## Objetivo

Criar um botão que abra a agenda do Google na tela de criação de eventos com os campos título, local e data já preenchidos.

## Ferramenta(s)
Android Studio v.4.1.3 - API 23: Android 6.0(Marshmallow)
